# IPL Score Prediction

## Overview

This project predicts the final score of an IPL match using machine learning. The model uses match information such as batting team, bowling team, current score, wickets, and overs to estimate the final total.

The goal of this project is to understand how machine learning models can be applied to sports data for prediction tasks.

## Dataset

The dataset contains historical IPL match data with features such as:

* Batting team
* Bowling team
* Current runs
* Wickets fallen
* Overs completed
* Runs scored in the last few overs

These features are used to train the model to predict the final score.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Machine Learning Models

The following regression models are used in the project:

* Linear Regression
* AdaBoost Regressor

These models are trained on historical data to learn patterns and estimate the final score.

## Project Workflow

1. Data collection and loading the dataset
2. Data preprocessing and cleaning
3. Feature selection and encoding
4. Training machine learning models
5. Evaluating model performance
6. Predicting the final score

## Purpose

This project was built as a beginner machine learning project to practice data preprocessing, model training, and prediction using Python.

## How to Run

1. Install the required Python libraries.
2. Run the Python script or Jupyter Notebook.
3. Train the model using the dataset.
4. Use the model to predict the final score based on match inputs.
